Football - Champions League

Computer Organisation, University of Toronto
Built with: Assembly

My teammate and I implemented a 2-player football game in hardware using an Alters DE2 board and light sensors. Completely coded in Assembly Language and C using several differnt types of interrupts including keyboard and sensors.

  • Key features included:
  • - Live goal updates on VGA display
  • - Automatic game timekeeping
  • - Game highlights displayed on Terminal
  • - Motorized kick functionality
