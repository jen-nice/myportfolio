Search Engine

Programming Languages, University of Toronto
Built with: Python, SQLite, Amazon Web Services, boto

I worked on the backend of the search engine which was coded in Python.

  • Key tasks I handled:
  • - Deployed the web application on Amazon Elastic Compute Cloud (EC2) from Amazon Web Services (AWS)
  • - Stored data generated by the crawler in an SQLite database
  • - Employed Boto the AWS SDK for Python to start the EC2 instances and create elastic IP addresses
  • - Analyzed the performance of the application on the server using Apache benchmarking tool, ab
  • - Coded Python scripts for launching and terminating the web application on AWS
  • - Monitored the status of the server such as utilization of CPU, memory, network, disk, and etc. using tools such as vmstat, mpstat, iostat and dstat
  • - Computed the PageRank score for each page that is visited by the crawler, given a list of URLs specified in a urls.txt file
