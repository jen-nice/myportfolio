Whitehead Agency Group Inc

I worked as a web developer for the Whitehead Agency Group Inc, mainly dealing with WordPress websites. A major project that I worked on was constructing the agency's new website. Given wireframes along with guidelines on the purpose of the pages, I customized site features to enhance the functionality and look of the website. Along with this I handled a few other client sites and was in charge of website backups, plugin updates and following good coding practices to maintain clean websites. Additionally, I also optimized website speed using best practices such as image compression, browser caching and minifying CSS and JS using recommended plugins.
